About this project

This website has been developed for the purposes of the course "Introduction to Computer Science". It is an e-shop of a coffee-makers enterprise. The website consists of 5 different pages developed using html5 and css. The purpose of the website is to provide a simple and efficient interaction with the user. This is achieved by incorporating graphic images and minimal commentation.
